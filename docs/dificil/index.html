Início

Pré Requisitos Máquinas Certificados Configuração Criptografia ETCD Controllers Workers Kubectl Rotas DNS Teste Dashboard Destruir

Neste tutorial vamos aprender como proceder na instalação de um cluster kubernetes usando o guia Kubernetes The Hard Way

Tomamos o cuidado de separar as partes do tutorial para facilitar o entendimento

Foram usados cores de fundo diferente para cada ambiente que os comandos serão executados, sendo:

Vermelho: comandos a serem executados na sua máquina local, notebook por exemplo
Verde: comandos a serem executados nas máquinas controller
Amarelo: comandos a serem executados nas máquinas worker

Pelo fato que iremos criar variaveis no shell na sua máquina local.
Essas variáveis seráo requisitadas durante várias partes do tutorial, então se faz necessário iniciar e terminar todo o tutorial usando a mesma sessão shell
por isso tire tempo para executar todo esse tutorial de uma única vez, pois retornar onde parou em outro dia ou outra sessão pode acarretar no tutorial ser mal sucedido.

Este tutorial tem carater de aprendizado, então não se esqueça no final de destruir as configurações na AWS para não ter surpresas desagradáveis

Preste bastante atenção nos comandos e suas ordens pois um erro poderá acarretar em recomeçar tudo de novo.
Caso tenha que recomeçar e as instâncias de máquinas criadas previamente ainda estiverem na lista mesmo que desligadas,
renome-as para outros nomes diferentes porque os scripts poderão se confundir
